User Population

The target user base for this pop-up is made up of students who are attempting to break their social media addiction. This user base can include students with and without mental illnesses, the intention that it will be a way to teach users how to interact with social media in a healthy way no matter what. The ideal test base would be people who believe they have an social media addiction so we can gauge how effective the users find using the pop up over a defined period of time.
